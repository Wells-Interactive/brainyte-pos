# Restaurant POS Project Specification

This folder structure is the blueprint for GitHub Copilot Agent.

## Objective
Develop a mobile-first Restaurant POS using PHP 8.2, MySQL, HTML5, CSS3 and Vanilla JavaScript.

### Core Modules
- Login
- Waiter
- Kitchen
- Bar
- API

### Features
- Waiter selects table
- Waiter adds menu items
- Running total updates instantly
- Send order
- Food routed automatically to Kitchen
- Drinks routed automatically to Bar
- Kitchen and Bar receive orders instantly via Server-Sent Events (SSE)
- No page refreshes
- PWA support
- Touch-friendly UI

### Database
users
restaurant_tables
menu_items
orders
order_items

### Roles
Waiter
Kitchen
Bar

### APIs
Login
